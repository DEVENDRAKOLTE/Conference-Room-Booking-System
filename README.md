# Conference Room Booking System

## 📌 Project Overview
The Conference Room Booking System is a full-stack web application developed to simplify conference room reservations within an organization. It allows users to book available rooms, manage reservations, and view booking schedules through a user-friendly interface.

## 🚀 Features
- User Login
- Book Conference Rooms
- View Room Availability
- Manage Reservations
- Admin Dashboard
- Responsive User Interface

## 🛠️ Technologies Used
### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Java
- Spring Boot
- Hibernate

### Database
- MySQL

## 📂 Project Structure
- Frontend
- Backend
- SQL Database Script

## ▶️ How to Run
1. Clone the repository.
2. Import the backend into Spring Tool Suite or IntelliJ IDEA.
3. Configure MySQL database.
4. Update database credentials in application.properties.
5. Run the Spring Boot application.
6. Open the frontend in your browser.
